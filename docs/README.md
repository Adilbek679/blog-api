## ERD (Entity-Relationship Diagram)

The homework requires an ERD image at **`docs/erd.png`** (or `erd.svg`), embedded in the project README.

1. **Source**: `erd.mmd` in this folder is a Mermaid diagram of the models (User, Category, Tag, Post, Comment).
2. **Export to image**:
   - Open [Mermaid Live Editor](https://mermaid.live), paste the contents of `erd.mmd`, then export as PNG/SVG, **or**
   - Use [dbdiagram.io](https://dbdiagram.io), draw.io, or any tool you prefer.
3. **Save** the image as `docs/erd.png` (or `docs/erd.svg`) so the main READMEâ€™s `![ER Diagram](docs/erd.png)` displays it.

Until `erd.png` exists, the README will show a broken image link.
